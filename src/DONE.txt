Функции с пометкой [b] могут работать не стабильно или привести к краху программы

Разворачивание при клике на иконку в трее. Иконка есть всегда, других функций нет.
При повторном запуске, разворачивается ранее запущенная копия если она свёрнута. Работает с копией в пределах папки.
Проверка системных библиотек нужных для работы прграммы, по умолчанию. Создаёт список с пометкой важности. Отключается sysini[LOG]CheckDll<>1 .

Индикация прав в главном окне. Без прав администратора часть функций может работать не правильно.
Изменение стиля окна, по умолчанию 0. sysini[GUI]Win7Style 0=тонкая рамка, 1=широкая рамка, расширенный стиль (только Win7) .

В панели главного окна:
[b] Список запущеных процессов. Показывает состояние на момент действия.
[b] Кнопки быстрого запуска. Вызывают соответствующие программ (Зависимы от главного окна и могут быть закрыты с ним)

[b] Во вкладках отображается вывод одного процесса. Предел текста ограничен (В случае краха процессы продолжат работу, но завершить их можно будет только через диспетчер, получить их вывод тоже нельзя, используйте лог. При перезапуске программы будут созданы новые копии процессов).
Установлено ограничение вывода, по умолчанию 600000. После чего окно вывода очищается с сохранением данных в файл, лимит можно уменьшить в sysini[GUI]ListingLimit



Возможность свернуть в трей вместо закрытия.
Запрос при закрытии программы. Окно запроса закрывается то таймауту.
